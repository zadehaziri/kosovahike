# Seed Database Script

Ky skript krijon të dhëna test për database-in e KosovaHike.

## Si të përdoret

```bash
cd hiking-backend
npm run seed
```

## Çfarë krijon

- **4 Përdorues** me kredencialet e login-it
- **5 Trails** me të dhëna test
- **2 Events** të planifikuar
- **2 Blogs** me artikuj test

## ⚠️ Kujdes

Ky skript **do të fshijë të gjitha të dhënat ekzistuese** në database! Përdore vetëm për development/testim.

## Kredencialet

Shiko skedarin `KREDENCIALET_LOGIN.md` në root të projektit për kredencialet e login-it.
