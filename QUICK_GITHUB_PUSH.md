## Step 1: Create New Repository on GitHub

1. Go to: https://github.com/new
2. Repository name: `kosovahike` (or any name you want)
3. Choose **Public**
4. **DON'T** check any boxes (no README, .gitignore, license)
5. Click **"Create repository"**

## Step 2: Copy Your New Repository URL

After creating, GitHub will show you commands. Copy this URL:
```
https://github.com/YOUR_USERNAME/kosovahike.git
```

## Step 3: Run These Commands (Copy & Paste)

```bash
# Add all files
git add .

# Commit
git commit -m "Initial commit: Full stack hiking trail management platform"

# Add your new repository (replace URL with your actual repo URL)
git remote add origin https://github.com/YOUR_USERNAME/kosovahike.git

# Push to GitHub
git push -u origin master
```
